HOW TO RUN

Reccomended to use a virtual environment

```shell
python3 -m venv .venv
```

Switch to the virtual environment

macOS/Linux

```shell
venv/bin/activate
```

Windows

```shell
venv\Scripts\Activate.ps1
```

Install the package

```shell
pip install .
```

Run app

```shell
streamlit run app.py
```
